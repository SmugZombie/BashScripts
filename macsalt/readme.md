A simple script made to act like salt / ancible to run a single command on multiple mac's at once.

<strong>Usage:</strong> <code>macsalt.sh all "whoami" -s</code>

<strong>Output:</strong><br>
soc1 - SUDO<br>
<strong>root</strong><br>
soc2 - SUDO<br>
<strong>root</strong><br>
soc3 - SUDO<br>
<strong>root</strong>
